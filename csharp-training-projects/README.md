.NET Framework, .NET Core C# training applications and sample codes.

* Image Viewer - Simple image viewer Windows Forms application - December 2018
* Chess Board - Designed a simple chess board with .NET Framework Windows Forms Application - December 2018

* TicTacToe - TicTacToe game implementation in C# - December 2018
* Product Catalog Demo - ADO.NET product catalog Windows Forms application using .NET Framework and Microsoft SQL Server 2017 (Udemy .NET and C# course recap project) - June 2019
* Inventory Service - Simple ASP.NET Core web application with REST API using .NET Core 2.1 - November 2019
